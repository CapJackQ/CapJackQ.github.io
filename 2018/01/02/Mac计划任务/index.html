Linux/Mac计划任务

发表于 2018-01-02 | 分类于 Linux

Linux crontab是用来定期执行程序的命令。crond命令每分锺会定期检查是否有要执行的工作，如果有要执行的工作便会自动执行该工作。

而linux任务调度的工作主要分为以下两类：

  • 系统执行的工作：系统周期性所要执行的工作，如备份系统数据、清理缓存
  • 个人执行的工作：某个用户定期要做的工作，例如每隔10分钟检查邮件服务器是否有新信，这些工作可由每个用户自行设置

语法

参数说明：

  • -e : 执行文字编辑器来设定时程表，内定的文字编辑器是 VI，如果你想用别的文字编辑器，则请先设定 VISUAL 环境变数来指定使用那个文字编辑器(比如说 setenv VISUAL joe)
  • -r : 删除目前的时程表
  • -l : 列出目前的时程表
  • 其中 p1 是表示分钟，p2 表示小时，p3 表示一个月份中的第几日，p4 表示月份，p5 表示一个星期中的第几天。program 表示要执行的程序。
  • 当 p1 为* 时表示每分钟都要执行 program，p2 为*时表示每小时都要执行程序，其余类推
  • 当 p1 为 a-b 时表示从第 a 分钟到第 b 分钟这段时间内要执行，p2 为 a-b 时表示从第 a 到第 b 小时都要执行，其余类推
  • 当 p1 为 */n 时表示每 n 分钟个时间间隔执行一次，p2 为 */n 表示每 n 小时个时间间隔执行一次，其余类推
  • 当 p1 为 a, b, c,… 时表示第 a, b, c,… 分钟要执行，p2 为 a, b, c,… 时表示第 a, b, c…个小时要执行，其余类推

示例

cron 表达式

# cron shell linux
