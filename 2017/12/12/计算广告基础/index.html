计算广告基础

发表于 2017-12-12 | 分类于 互联网广告 计算广告

出发点：在互联网广告中，计算之所以可以发挥巨大的作用，与它的根本技术特点有关。可衡量的效果以及相应的计算优化是在线广告的主要特点。千次展示期望收入(expected Cost Per Mille，eCPM)正是计算广告中最为核心的量化指标之一。与广告的信息传达过程相关，eCPM可以分解为点击率和点击价值的乘积。

广告的有效性原理

广告的信息接收过程分为选择(selection)、解释(interpretation)与态度(attitude)三个大阶段，或者进一步分解为曝光(exposure)、关注(attention)、理解(comprehension)、接受(acceptance)、保持(retention)与决策(decision)6个子阶段。

  1. 曝光(exposure)阶段。这一阶段是广告物理上展现出来的过程，此阶段的有效程度往往与广告位的物理属性有关，并没有技术优化空间。
  2. 关注(attention)阶段。这一阶段是指受众从物理上接触到广告到意识上注意到它的过程。使关注阶段效率提高：a、尽量不打断用户任务。b、明确传达向用户推送词广告的原因，内容要符合用户兴趣或需求(行为定向)。
  3. 理解(comprehension)阶段。原则：a、广告内容要在用户能理解的具体兴趣范围内，必须真正的做到精准受众定向。b、要注意设定与关注程度相匹配的理解门槛。
  4. 接受(acceptance)阶段。广告的上下文环境对广告的接受程度有着很大的影响，让合适的广告出现在合适的媒体上(ad safety)。
  5. 保持(retention)阶段。
  6. 决策(decision)阶段

定性地说，越靠前的阶段，其效果的改善对点击率的贡献越大；而越靠后的阶段，其效果的改善对转化率的贡献越大。但并非孤立和绝对。

互联网广告的技术特点

  1. 技术和计算导向。在线广告可以进行精细的受众定向，而技术又使得广告决策和交易朝着计算驱动的方向发展。
  2. 效果的可衡量性。在一个特定时期不同广告和算法表现出来的差异更有意义。
  3. 倡议和投放方式的标准化。标准化的驱动力来自于受众定向与程序购买。
  4. 媒体概念的多样化
  5. 数据驱动的投放决策。互联网化的根本驱动力可以认为是数据的深入加工和利用。

计算广告的核心问题

核心问题：为一系列用户与环境的组合找到最合适的广告投放策略以优化整体广告活动的利润。

广告优化的是一组展示上的效果，而非孤立的某一次展示上的效果。优化结果是“投放策略”，不一定是具体的广告。

广告收入的分解

对一个广告市场中具体的产品形态，往往能够主动优化的是收入而非成本。广告点击与广告展现的比率称为点击率(Click Through Rate，CTR)；点击行为成功后，将会打开广告主的落地页(landing page)，落地页成功打开次数与点击次数的比率称为到达率；如果用户从落地页开始，进一步完成下单等操作，则称为转化，转化次数与到达次数的比例称为转化率(Conversion Rate，CVR)。

结算方式与eCPM估计的关系

大多数广告需要计算给定(a，u，c)三元组的eCPM的可以进行决策。

  1. CPM结算，即按照千次展示结算。这种方式由供给方和需求方预定好千次展示的计费标准，至于能否带来相应收益，由需求方估计和控制，
  2. CPC结算，即按点击结算。点击率估计交由供给方完成。点击价值的估计由需求方完成，并通过点击出价的方式向市场通知估价；供给方通过收集大量用户行为数据可以相对准确地估计点击率。CPC结算在效果类广告中比较出众。
  3. CPS(cost per sale)/CPA(cost per action)/ROI结算，按照销售订单数、转化行为数或投入产出比来结算。按照转化付费的变相。
  4. CPT(cost per time)结算，针对大品牌广告主特定的广告活动。

结算方式 点击率估计 点击价值估计 优缺点 使用场景
CPT 需求方 需求方 可以充分发挥橱窗效应；无法利用手中定向技术 高曝光品牌广告
